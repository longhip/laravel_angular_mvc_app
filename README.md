## Steedoffice Core (MVC).

Author: longhip.dev@gmail.com

## Clone

```sh
$ git clone 
```

## Composer update dependencies

```sh
$ cd /path/to/project
```
```sh
$ cd api
```
```sh
$ composer update
```

## Connect mysql
You can connect with .env file

## Migrate and Seeder

```sh
$ cd /path/to/project
```
```sh
$ cd api
```
```sh
$ php artisan migrate
```
```sh
$ php artisan db:seed
```
